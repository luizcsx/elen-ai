# Guia de Contribuição

Obrigado pelo interesse em colaborar com o Élen. Este documento explica como você pode ajudar o projeto a crescer de forma organizada e respeitosa.

## Código de Conduta

Antes de tudo, pedimos que mantenha um ambiente acolhedor. Seja respeitoso com outros colaboradores, evite linguagem ofensiva e lembre-se de que pessoas de diferentes origens participam deste projeto.

## Como posso contribuir?

Existem várias formas de participar:

- **Relatando problemas**: encontrou algo estranho? Abra uma issue descrevendo o que aconteceu, como reproduzir e qual comportamento você esperava.
- **Sugerindo melhorias**: ideias para novas funcionalidades ou mudanças na interface são bem-vindas. Explique o porquê da sugestão.
- **Corrigindo bugs ou adicionando código**: veja a seção abaixo sobre como enviar um Pull Request.
- **Melhorando a documentação**: correções de digitação, traduções ou explicações mais claras também contam.
